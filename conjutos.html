Conjuntos Numericos

A definição para conjunto é a mesma usada na linguagem comum: um agrupamento, uma coleção. Cada membro que entra na formação do conjunto é chamado de elemento.

Exemplos de Conjuntos

1) O conjunto das vogais: A = { x | x é uma vogal} ou A = {a, e, i, o, u}

2) O conjunto dos números ímpares positivos: B = { x | x é um número ímpar positivo} ou B = {1, 3, 5, 7, 9, ...}

3) O conjunto dos naipes das cartas de um baralho: C = { x | x é um naipe} ou C = {ouro, copas, espada e paus}


Existe o simbolo, pertence e o não pertence, eles são representados por esses respectivos simbolos: ∈, ∉

Exemplo: U pertence ao conjunto A. Ou, u ∈ A

Exemplo: 9 pertence ao conjunto B. Ou, 9 ∈ B

Exemplo: 15 não pertence ao conjunto C. Ou, 15 ∉ C

Conjunto é um grupo seleto de coisas, como numeros pares ou ate mesmo números negativos.



Também temo os simbolos Está Contido ⊂, e Não Está Contido ⊄

Conjunto A = 1, 2, 3, 4, 5, 6, 7, 8

Conjunto B = 3, 4, 5

Conjunto C = 9, 10, 11, 12

Exemplo : B está contido em A, ou seja, B ⊂ A

Exemplo : C não esta contido em A, ou seja, C ⊄ A



Também temos os símbolo de União ∩ ∪

Intersecção

Quando os conjuntos tem alguma relação usa-se esse simbolo ∩

Exemplo: A = {1,2,3,4,5} e B {4,5,6,7} ou seja A ∩ B = {4,5}

Diferença

A diferença entre dois conjuntos: Exemplos: Sejam 𝐴 = {1,2, 3, 4, 5} 𝑒 𝐵 = { 2, 3}, 𝐴 − 𝐵 = {1, 4, 5}

Sejam 𝐴 = {−2, 3, 6, 9, 10, 20, 55} 𝑒 𝐵 = { 0, 1, 3, 9, 20, 60}, 𝐴 − 𝐵 = {−2, 6, 10, 55}

Formalmente, o conjunto diferença entre A e B é dado por 𝐴 − 𝐵 = {𝑥 | 𝑥 ∈ 𝐴 𝑒 𝑥 ∉ 𝐵}
